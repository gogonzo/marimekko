# mekko: Marimekko and Mosaic Plots for 'ggplot2'

Create Marimekko (mosaic) plots as a 'ggplot2' layer. Column widths
encode marginal proportions of one categorical variable and segment
heights encode conditional proportions of a second categorical variable.
Provides a simpler alternative to 'ggmosaic'.
