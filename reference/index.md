# Package index

## Plot layers

Core geom and stat functions for building Marimekko plots.

- [`StatMarimekko`](geom_mekko.md) [`geom_mekko()`](geom_mekko.md)
  [`GeomMarimekko`](geom_mekko.md) : Create a Marimekko (mosaic) plot
- [`geom_mekko_text()`](geom_mekko_text.md) : Add text labels to a
  Marimekko plot
- [`geom_mekko_label()`](geom_mekko_label.md) : Add labels with
  background to a Marimekko plot
- [`geom_mekko_jitter()`](geom_mekko_jitter.md)
  [`StatMarimekkoJitter`](geom_mekko_jitter.md) : Jitter individual
  observations within Marimekko tiles
- [`geom_mekko_multi()`](geom_mekko_multi.md)
  [`StatMarimekkoMulti`](geom_mekko_multi.md) : Multi-variable Marimekko
  plot with nested splits

## Scales

Custom scales that label axes with category names.

- [`scale_x_mekko()`](scale_x_mekko.md) : X-axis scale for Marimekko
  plots
- [`scale_y_mekko()`](scale_y_mekko.md) : Y-axis scale for Marimekko
  plots

## Annotations

Annotations for statistical tests.

- [`annotate_chisq()`](annotate_chisq.md) : Annotate a Marimekko plot
  with a chi-squared test

## Utilities

Helper functions for data extraction and theming.

- [`fortify_mekko()`](fortify_mekko.md) : Compute Marimekko tile
  rectangles as a data frame
- [`theme_mekko()`](theme_mekko.md) : Minimal theme for Marimekko plots
