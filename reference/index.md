# Package index

## Plot layers

Core geom and stat functions for building marimekko plots.

- [`geom_marimekko()`](geom_marimekko.md) : Generalized mosaic plot with
  formula-based variable nesting
- [`geom_marimekko_text()`](geom_marimekko_text.md) : Add text labels to
  a marimekko plot
- [`geom_marimekko_label()`](geom_marimekko_label.md) : Add labels with
  background to a marimekko plot
- [`StatMarimekkoTiles`](StatMarimekkoTiles.md) : Retrieve computed tile
  positions from a marimekko layer

## Utilities

Helper functions for data extraction and theming.

- [`fortify_marimekko()`](fortify_marimekko.md) : Compute marimekko tile
  rectangles as a data frame
- [`marimekko_pal`](marimekko_pal.md) : Unikko-inspired colour palette
- [`theme_marimekko()`](theme_marimekko.md) : Minimal theme for
  marimekko plots
